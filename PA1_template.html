Imputing missing values

  1. Calculate and report the total number of missing values in the dataset
na_steps<-sum(is.na(activity$steps))
na_date<-sum(is.na(activity$date))
na_interval<-sum(is.na(activity$interval))

  1. Devise a strategy for filling in all of the missing values in the dataset.
    During 61 days, corresponding to a significant sample, we can expect a repeat of values for a same intervals. The nearest behavior for an interval is that which corresponds to the average of the same interval for the other days. So, we will fill the missing values in the dataset with the mean for that 5-minute interval

  2. Create a new dataset that is equal to the original dataset but with the missing data filled in.

newactivity <- activity
l=length(newactivity[,1])
for(i in 1:l)
  { 
  if(is.na(newactivity$steps[i]))
    {
    newactivity$steps[i]<-StepsByInterval$meanSteps[StepsByInterval$interval==newactivity$interval[i]]
    }
  }
